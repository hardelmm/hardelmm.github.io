mysql常用查询指令

2017-07-12
约 1006 字 - 预计阅读 3 分钟 | 阅读

1、查看数据库连接池最大连接数

1
2

show variables like '%max_connections%';
#实际MySQL服务器允许的最大连接数16384；

2、设置数据库连接池最大连接数

1

set global max_connections=1000;

3、查看process信息

如果是root帐号，能看到所有用户的当前连接。如果是其它普通帐号，只能看到自己占用的连接。

1
2

show processlist; #显示前100条
show full processlist; #全部列出

5、查看数据库版本

1

select @@Version;

6、查看表状态属性

1

show table status like '表名';

7、更改表引擎属性

1

ALTER TABLE test_table ENGINE=MyISAM;

8、查看数据库状态

1

show status;

show status like '%Aborted_clients%';	#由于客户没有正确关闭连接已经死掉，已经放弃的连接数量。
show status like '%Aborted_connects%';	#尝试已经失败的MySQL服务器的连接的次数。
show status like '%Connections%';		#试图连接MySQL服务器的次数。
show status like '%Created_tmp_tables%'; #当执行语句时，已经被创造了的隐含临时表的数量。
show status like '%Delayed_insert_threads%'; #正在使用的延迟插入处理器线程的数量。
show status like '%Delayed_writes%';	#用INSERT DELAYED写入的行数。 
show status like '%Delayed_errors%';	#用INSERT DELAYED写入的发生某些错误(可能重复键值)的行数。
show status like '%Flush_commands%';	#执行FLUSH命令的次数。
show status like '%Handler_delete%';	#请求从一张表中删除行的次数。
show status like '%Handler_read_first%';#请求读入表中第一行的次数。
show status like '%Handler_read_key%';	#请求数字基于键读行。
show status like '%Handler_read_next%';	#请求读入基于一个键的一行的次数。
show status like '%Handler_read_rnd%';	#请求读入基于一个固定位置的一行的次数。
show status like '%Handler_update%';	#请求更新表中一行的次数。
show status like '%Handler_write%';		#请求向表中插入一行的次数。
show status like '%Key_blocks_used%';	#用于关键字缓存的块的数量。
show status like '%Key_read_requests%';	#请求从缓存读入一个键值的次数。
show status like '%Key_reads%';			#从磁盘物理读入一个键值的次数。
show status like '%Key_write_requests%'; #请求将一个关键字块写入缓存次数。
show status like '%Key_writes%';		#将一个键值块物理写入磁盘的次数。
show status like '%Max_used_connections%'; #同时使用的连接的最大数目。
show status like '%Not_flushed_key_blocks%'; #在键缓存中已经改变但是还没被清空到磁盘上的键块。
show status like '%Not_flushed_delayed_rows%'; #在INSERT DELAY队列中等待写入的行的数量。
show status like '%Open_tables%';		#打开表的数量。
show status like '%Open_files%';		#打开文件的数量。
show status like '%Open_streams%';		#打开流的数量(主要用于日志记载）
show status like '%Opened_tables%';		#已经打开的表的数量。
show status like '%Questions%';			#发往服务器的查询的数量。
show status like '%Slow_queries%';		#要花超过long_query_time时间的查询数量。
show status like '%Threads_connected%';	#当前打开的连接的数量。
show status like '%Threads_running%';	#不在睡眠的线程数量。
show status like '%Uptime%';			#服务器工作了多少秒。
